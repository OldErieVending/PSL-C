# PSL-C

This repository hosts the scientific paper describing PSL-C (Prompt Structure Language – Cinematic), 
a domain-specific language for structured, reproducible text-to-video prompting.

Author: Zach Bogart  
PromptBuilder.us
Clyde, NY

Paper: PSL-C_Full_Scientific_Paper_Zach_Bogart.pdf
